Público objetivo

Dado que estamos tratando con un sitio web de caracter gubernamental (debido que estamos hablando de la web del Ayuntamiento de Almería), tenemos que tener en cuenta que el público objetivo será diverso, ya que podremos encontrar:

  • Turistas: usarán el sitio web para informarse de los eventos culturales o musicales, sitios de interés, alojamiento, transporte público, etc. Se debería dedicar una página específica a esto, donde rediccionaremos a ese tipo de público desde la propia zona de navegación.
  • Residentes locales: usarán el sitio web para encontrar información sobre los servicios municipales, trámites administrativos, noticias de la ciudad, etc. Será el público mayoritario.
  • Estudiantes/Padres de estudiantes: usarán el sitio web para encontrar información sobre las escuelas, institutos y universidades cercanas a la localidad.
  • Empresas y Autónomos: usarán el sitio web para encontrar información sobre las distintas licencias comerciales, permisos de construcción, regulaciones comerciales, ofertas de licitación, etc.
  • Busqueda de empleo: usarán el sitio web para encontrar información sobre las distintas ofertas públicas de empleo, cursos de capacitación, etc.
  • Medios de comunicación: usarán el sitio web para informarse sobre los últimos comunicados de prensas oficiales del ayuntamiento, etc.
  • Personas interesadas por la ciudad: usarán el sitio web para informarse sobre la ciudad de Almería, por motu proprio o por motivos profesionales

Tipo de web

El sitio web es un ejemplo de sitio web gubernamental o municipal. Estos sitios son una plataforma importante para la comunicación, prestación de servicios y la participación ciudadana, donde su objetivo principal es facilitar el acceso a la información relevante y los servicios ofrecidos por la administración municipal

Elementos de diseño

Puesto que estamos tratando con un sitio web con una importancia bastante relevante (estamos hablando de un sitio gubernamental), deberemos añadir una serie de elementos de diseño que nos permitan generar un ambiente cómodo para todos los tipos y perfiles de usuarios. Algunos de los elementos de diseño donde quiero centrarme son:

  • Líneas: separar el contenido de forma que el usuario pueda apreciar las distintas secciones de forma clara.
  • Barra de busqueda: debido a que los usuarios necesitarán acceder a secciones/páginas concretas, deberemos brindarle al usuario este elemento para que encuentre la información de forma fácil y sencilla.
  • Menú de navegación: estructurar el menú de navegaciónn de forma clara, facilitando así el acceso a las diferentes secciones del sitio.
  • Diseño responsivo: como gran parte de los usuarios accederán al sitio web desde dispositivos móviles, deberemos diseñar el sitio pensando en su adaptabilidad a dichos dispositivos
  • Enlaces rápidos: debido a que gran parte de usuarios accederán al sitio web para realizar un mero trámite, deberemos crear pequeños "atajos" en nuestro sitio para que accedan a esos lugares cómunes de forma rápida
